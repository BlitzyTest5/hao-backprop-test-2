# hao-backprop-test-2
Test project for backprop integration. Do not touch!

## Description
This is a minimal HTTP server that returns "Hello, World!" for all requests. It serves as a controlled environment for backprop integration testing.

## Requirements
- Python 3.7 or higher

## Installation
```
pip install -r requirements.txt
```

## Running the server
```
python app.py
```

## Verification
After starting the server, it will be available at http://127.0.0.1:3000
All requests to this endpoint will return "Hello, World!" with a 200 status code.
